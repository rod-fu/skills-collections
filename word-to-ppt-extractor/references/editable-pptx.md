# 可编辑 PPTX 转译硬约束

当用户明确要求“在 PowerPoint 或 Keynote 中继续编辑文字位置”时，才生成 PPTX。PPTX 不是 HTML 的像素级复刻，而是“可编辑优先”的降级版本。

## 4 条硬约束

1. **纯文字流布局**
   - 使用标题、段落、列表、表格和基础形状表达页面。
   - 避免 `position:absolute`、`float`、复杂 `transform`、深层嵌套 Grid。

2. **边框与背景限制**
   - 只使用纯色背景、简单矩形、线条、圆形和基础箭头。
   - 渐变、玻璃拟态、滤镜、混合模式、遮罩和复杂阴影必须降级。

3. **图表与视觉母题可编辑**
   - 优先用基础形状和文本框重建图解。
   - 复杂 SVG、Canvas、CSS 动画应导出为静态图片，并在备注中标明不可编辑部分。

4. **版式可预期**
   - 使用固定 16:9 画布。
   - 字体层级不超过 4 级。
   - 每页主体元素不超过 12 个。
   - 页面标题、页码、备注位置保持一致。

## 降级说明模板

```markdown
### PPTX 排版降级说明
为保证 PowerPoint 内可编辑，以下 HTML 效果已降级：
- [效果] -> [降级方式]
- [不可编辑元素] -> [保留为静态图片/基础形状]
建议以 HTML 演示版或 PDF 作为高保真展示版本，以 PPTX 作为会后编辑版本。
```
